-Div

La etiqueta Div define una división o una sección en un documento HTML. Se usa frecuentemente como una sección que contiene otros elementos HTML para modificarlos con CSS o realizar ciertas tareas con JavaScript. <div style="background-color:#b3e6ff"> <h3>This is a heading</h3> <p>This is a paragraph.</p> </div>

"Nuestro primer modal"

Un modal es un cuadro de diálogo/ventana emergente que se muestra por encima de la página actual.
A continuación nos encontramos con un modal creado con Bootstrap, evitando así usar JavaScrip para su utilización.

<!-- Trigger the modal with a button --> <button type="button" class="btn btn-info btn-lg" data-toggle="modal" data-target="#myModal">Open Modal</button> <!-- Modal --> <div id="myModal" class="modal fade" role="dialog"> <div class="modal-dialog"> <!-- Modal content--> <div class="modal-content"> <div class="modal-header"> <button type="button" class="close" data-dismiss="modal">&times;</button> <h4 class="modal-title"> ¡Atención!</h4> </div> <div class="modal-body"> <p> Texto del modal aquí. </p> </div> <div class="modal-footer"> <button type="button" class="btn btn-default" data-dismiss="modal">Close</button> </div> </div> </div> </div>
