CSS3

"Conociendo las clases" Plantilla básica de la definición un conjunto de estilos css

-Selectores:

Los selectores de CSS se usan para "encontrar" (o seleccionar) elementos de HTML basados en sus elementos de nombre, id, clases, atributos, etc.. Algunos de los selectores más básicos, son los siguientes:

  • Selector "elemento":
    Selecciona todos los elementos con el nombre del elemento especificado.
    Syntax: element { css declarations; } Ejemplo:
    Selecciona y estiliza todos los elementos <p> de la página, cambiando, en este caso, el color de las letras que esten contenidas en el elemento a azul. p { color: blue; }

  • Selector "elemento, elemento":
    Selecciona varios los elementos y los estiliza a todos de igual manera. Los elementos se separan con una coma.
    Syntax: element, element { css declarations; } Ejemplo:
    Selecciona y estiliza todos los elementos <p> y los elementos <h1> de la página, cambiando, en este caso, el color de las letras que esten contenidas en los elementos a azul. p, h1 { color: blue; }

  • Selector "elemento elemento":
    Selecciona elementos detro de otros elementos.
    Syntax: element element { css declarations; } Ejemplo:
    Selecciona y estiliza todos los elementos <p> que estén contenidos en elementos <div> de la página, cambiando, en este caso, el color de las letras a azul. div p { color: blue; }

  • Selector "id":
    Estiliza los elementos que contengan el id especificado con # en el archivo css.
    Syntax: #id { css declarations; } Ejemplo:
    En este caso, el estilo "background-color: yellow" solo se aplicará al elemento que tiene cómo id "fisrtname". CSS: #firstname { background-color: yellow; } HTML <p id="firstname">My name is Donald.</p> <p id="hometown">I live in Duckburg.</p>
  • Selector "class":
    El selector ".class" selecciona elementos que tengan este atributo especificado. Para ello, escribe "." seguido del nombre de la clase a la que quieres asignar un estilo. Puedes tambien afectar a un elemento específico escribiendo dicho elemento, seguido de "." y el nombre de la clase.
    Syntax: .class { css declarations; } Ejemplo1:
    En este caso el color de fondo del elemento <p> que contiene la clase "center" será azul. CSS: .center { background-color blue; } HTML: <p class="center">This is a paragraph.</p> <p>This is other paragraph.</p> Ejemplo2:
    Así pues, cómo hemos mencionado antes podemos estilizar un elemento en concreto que contenga la clase, dejando fuera otro tipo de elementos que tengan la misma clase. element.class { css declarations; } CSS: p.center { background-color: blue; } HTML: <p class="center">This is a paragraph.</p> <div class="center">This paragraph is in another element.</div>

-Margin:

Esta propiedad se usa para crear espacio al rededor de elementos, por fuera de cualquier borde definido. Se tiene control total sobre los margenes gracias a las propiedades para establecer margenes concretos a cada uno de los bordes del elemento (top, right, bottom, and left).
Ejemplo: CSS: .margin-ej2 { margin: 70px; border: 5px solid lightblue; } HTML: <div class="margin-ej2">This element has a margin of 70px.</div>
This element has a margin of 70px.

-Padding:

Esta propiedad se usa para general espacio al rededor del contenido de un elemento, por dentro de cualquier borde definido. Se tiene control total sobre el padding gracias a las propiedades para establecer el padding a cada uno de los bordes del elemento (top, right, bottom, and left).
Ejemplo: CSS: .padding-ej2 { padding: 70px; border: 5px solid lightblue; } HTML: <div class="padding-ej2">This element has a margin of 70px.</div>
This element has a margin of 70px.

"Media queries"

Ejemplo del uso de distintas configuraciones de estilos según la resolución de la pantalla en la que se vea:

CSS: .media-ej2 { background-color: blue; color: blue; } @media only screen and (max-width: 600px) { .media-ej2 { background-color: lightblue; } } HTML: <div class="media-ej2"> <h1>The @media Rule</h1> <p>Resize the browser window. When the width of this document is 600 pixels or less, the background-color is "lightblue", otherwise it is "blue".</p> </div>

The @media Rule
